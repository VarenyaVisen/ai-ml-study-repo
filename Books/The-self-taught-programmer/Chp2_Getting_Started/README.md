#### What is programming?
Programming is written instructions for a computer to execute, the instruction might tell computer to print HELLO WORLD, scrape data from internet or read contents of a file and save it to a database. These instructions are called `code`

earlier programmer used to write in `low level programming language` it is closer to being written in binary 0s and 1s now there are `high level programming language` which is easier to understand and read like english
