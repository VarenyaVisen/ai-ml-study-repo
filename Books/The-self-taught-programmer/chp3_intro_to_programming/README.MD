### Chapter 3 - Introduction to programming

#### comments
A comment is a line of code written in English or any language preceded by a speacial symbol `#` which tells the programming language to ignore that line , a comment explains what a line of code does. Programmers use comments to clarify why they did something to make the line of code easier to understand 

#### Keywords
the python language has a list of words with special meaning. These are called keywords, `for` is a keyword for example
note- we cannot use keywords as a variable name 

#### spacing 
this is indentation which helps python to know when code blocks begin and end

#### Data Types 
different kind of data in python are grouped into different categories called data types. In python each data value, like 2 or "Hello world" is called and `object`
think of an `OBJECT` as a data value in python with 3 properties: an identity, a data type and a value

- an identity of an object is where it is stored in memory, which never changes
- the data type of an object is the category of data the object belongs to and determines the properties the object has, this also never changes
- The value of an object is the data it represents
